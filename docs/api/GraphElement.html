GraphElement(elementID)

A class that represents any positioned, unique object in a graph. Usually this is not instantiated, but rather extended. Look at NodeElement and EdgeElement as examples.

getHashString(usePositionopt) → {String}

Computes the hash string that represents this element and its current state uniquely and deterministically. NodeGraph uses this to compute its hash code that distinguishes it from other graphs. The generated string should only be used to differentiate between other objects of the same hash function. In other words, every instance should only be compared to other instances of the same class (or extended class without override).
Parameters:
Name Type Attributes Default Description
usePosition Boolean <optional>
 true Whether to consider positioning as part of the unique state.
